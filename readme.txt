MailMiner
=========

Description:
------------
MailMiner is a Python tool designed to search for email addresses of a target domain. It utilizes web scraping techniques to extract email addresses from web pages linked to the provided URL. This tool can be helpful for gathering contact information or performing reconnaissance during security assessments.

Features:
---------
- Search for email addresses of the target domain.
- Set the maximum number of URLs to process.
- Extract email addresses from web pages.
- Prompt user for target URL and maximum URLs interactively.

Usage:
------
1. Run the script `mail_miner.py` using Python 3.x.
2. Use the `-h` or `--help` parameter to display the help message and command-line options.
3. Follow the prompts to enter the target URL and maximum number of URLs to process.
4. Wait for the tool to search and extract email addresses.
5. View the list of extracted email addresses.

Command-line Arguments:
------------------------
- `--host`, `-u`: URL to scan (e.g., https://www.example.com/).
- `--maxurls`, `-murls`: Maximum number of URLs to process (default is 100).
- `-h`, `--help`: Display help message and command-line options.

Note: If no arguments are provided, the script will prompt the user to enter the required information interactively.
